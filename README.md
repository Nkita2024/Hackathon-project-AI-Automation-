# 🚀 AI Automation Hackathon Project

## 📖 Overview
This project demonstrates a **full‑stack AI automation tool** that scrapes company websites, extracts key business information, and enriches it with **AI‑powered insights**.  
It showcases skills in **FastAPI**, **Playwright (for scraping JS‑heavy sites)**, **OpenAI API integration**, and **cloud deployment**.

---

## 🎯 Goals
- Demonstrate **backend engineering** with FastAPI.
- Show **web scraping automation** using Playwright.
- Highlight **AI integration** with OpenAI’s GPT models.
- Provide a **live demo** deployable on Render/Vercel for interview review.

---

## ✨ Features
- **Scraping Helper** → Extracts title, emails, phones, addresses, and text from websites.
- **AI Enrichment** → Converts raw text into structured business insights.
- **REST API Endpoints**:
  - `POST /enrich` → Scrape + enrich a given URL.
  - `GET /results` → Retrieve all processed results.
  - `GET /` → Serves `index.html` frontend.
- **Frontend** → Simple HTML interface for testing.

---

## 🛠️ Tech Stack
- **Backend**: FastAPI
- **Scraping**: Playwright + BeautifulSoup
- **AI Layer**: OpenAI GPT (ChatCompletion API)
- **Frontend**: HTML
- **Deployment**: Vercel
